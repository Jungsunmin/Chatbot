## 사용 전 검토 필요

- `source_url`이 공식 안내 페이지인지 확인
- `updated_at`을 실제 검토 날짜 또는 원문 업데이트 기준으로 확정
- `curation_status`를 검토 후 `human_curated` 또는 `reviewed`로 변경
- 수수료, 제출서류, 기한은 변경 가능성이 높으므로 최신 공지와 대조
