# 서비스 목표 정의

**승인일**: 2026-06-02 · **범위**: KU 외국인 유학생용 **RAG FAQ 챗봇** (출처 필수)

## 문제 정의

건국대학교(KU) **외국인 유학생**은 **언어 장벽**과 함께 가이드북·국제처·외국인학생센터 안내가 **PDF·웹·공지 등으로 분산**되어 있다. 비자, 등록, 수강, 기숙사, 캠퍼스 생활 등 **행정·생활 FAQ**를 빠르게 찾기 어렵고, 국제처에 **반복 문의**가 발생하거나 **잘못된 정보**에 의존할 수 있다.

## 대상 사용자


| 구분     | 설명                                                |
| ------ | ------------------------------------------------- |
| **1차** | KU **외국인 유학생**(학부·대학원·어학당·교환 포함) — 챗봇으로 FAQ 셀프서비스 |
| **2차** | 국제처·외국인학생센터 **행정 담당자**(공식 소스 제공·검수 — MVP 이후 본격)   |


## 가치 제안

- KU **가이드북·FAQ 소스**를 RAG로 검색하는 **챗봇**으로, 질문에 **출처(citation)와 함께** 답변한다.
- PDF·웹·국제처 **1:1 문의 전 1차 셀프서비스** 진입점을 제공한다.
- **환각 완화**: 근거 없는 행정 답변을 내지 않고, 소스에 없으면 FAQ-only 또는 안내 문구로 처리한다.
- **한국어·영어·중국어·일본어(ko/en/zh/ja)** 질의·응답·UI — 소스·번역 우선순위는 후속 Skill에서 확정.

## 성공 기준


| ID   | 기준                                 | 측정(초안)                  |
| ---- | ---------------------------------- | ----------------------- |
| SC-1 | 가이드북·FAQ **소스 기반** 답변 + **출처 표시**  | 샘플 질의 세트에서 citation 포함률 |
| SC-2 | 핵심 FAQ 카테고리(비자·등록·수강·기숙사 등) **커버** | 카테고리별 샘플 Q&A 통과율        |
| SC-3 | **환각·무출처** 행정 답변 최소화               | 오답·무근거 응답 비율(수동 QA)     |
| SC-4 | (선택) **4개 언어** 질의·응답 또는 UI         | i18n·소스 번역 커버리지         |


## 비목표 (서비스 전체 관점)

- **풀앱** — 맞춤 대시보드, Push, 캠퍼스 지도, 커뮤니티·룸메이트, 통합 알림 센터
- 타 대학·타 캠퍼스 범용 플랫폼
- 학교 **공식 성적·등록 시스템 대체**(포털 SSO·성적 API)
- **출처 없는 AI** 행정 답변
- MVP 단계 **국제처 공식 승인·운영 위임** 전제
