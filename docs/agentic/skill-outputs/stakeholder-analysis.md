| **소스·콘텐츠 담당** (팀 또는 국제처 위임) | `data/sources` **`.md` 정확·최신** 유지 | 섹션 분량·4언어 번역·reindex 부담 | 소스 작성·frontmatter(`title`, `source_url`, `lang`), **인덱스 갱신** | **오래된 소스** → 잘못된 citation |
| **번역·i18n 담당** (콘텐츠 담당과 동일 가능) | **ko/en/zh/ja** 질의·답변·UI 문구 품질 | 용어 불일치, 부분 번역만 존재 | UI 문자열·소스 `lang` 필드·샘플 Q&A QA | 한 언어만 완성 → **다른 언어 사용자 이탈** |
| **개발 팀** | RAG 챗봇 **프로토타입 검증**·유지보수 | 범위 과다, LLM 품질·Mac/GPU 환경, 환각 완화 | FastAPI·Chroma·HF LLM, Expo 챗 UI, `/chat`·citation·unknown 정책 | 기술 부채, **faithfulness** 미달 |

## MVP에서 직접 관여하지 않음 (Non-Scope 연결)

| Stakeholder (풀앱·2차) | Skill 1 Non-Goals와의 관계 |
|------------------------|------------------------------|
| **커뮤니티·룸메이트 사용자** | 커뮤니티·매칭 — 챗봇 MVP 제외 |
| **Push·학사 일정 운영** | Push·대시보드 — 제외 |
| **캠퍼스 지도·시설** | 지도·길찾기 — 제외 |
| **학교 IT·포털 (SSO)** | 포털·성적 대체 — 제외; 챗봇 MVP는 **게스트/무로그인** 가능 전제 |

후속 Skill(`requirements-decomposition`, `mvp-scope-planning`)에서 위 역할을 요구사항에 **넣지 않음**.

## 다국어(ko / en / zh / ja) — 이해관계자 관점

| **개발** | API `lang` 필드, i18n 문자열, 임베딩·LLM 다국어 지원 검증 |
