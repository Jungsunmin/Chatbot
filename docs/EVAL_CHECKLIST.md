# 평가 체크리스트 (RAG MVP)

## 자동 (보조)

- [ ] `build_index.py` 후 chunk 수 > 0
- [ ] 테스트 질문 10개에 retrieval 결과 존재

## 휴먼 (필수)

각 테스트 질문에 대해:

- [ ] 답이 **제공 FAQ와 일치**하는가
- [ ] **없는 날짜·연락처**를 지어내지 않았는가
- [ ] `citations` 의 `source_uri` 가 실제 문서인가
- [ ] clarify 후 답이 **슬롯(학기·학생구분 등)** 과 맞는가

## fallback

- [ ] 검색 결과 없을 때 국제처 안내 문구 표시

LoRA 단계는 이 체크리스트 통과 후 진행.
