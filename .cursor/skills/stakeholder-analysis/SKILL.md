# Stakeholder Analysis

## When to use

- 요구사항 도출 전 이해관계자 정렬
- 국제처·학생·개발·운영 역할 경계가 불명확할 때

## Prerequisites

- `service-goal-definition` 완료 (`skill_outputs.md`)

## Procedure

1. 직접/간접 이해관계자 목록 작성
2. 표: Goal, Pain Point, Responsibility, Risk
3. MVP에서 **관여하지 않는** 이해관계자는 Non-Scope에 연결

## Output template

`docs/agentic/skill_outputs.md` → `## stakeholder-analysis`

```md
# Stakeholder Analysis
| Stakeholder | Goal | Pain Point | Responsibility | Risk |
```
