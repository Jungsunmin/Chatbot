# Documentation Handoff

## When to use

- 스프린트·MVP 마일스톤 종료
- 국제처·팀 인수인계

## Procedure

1. `context_packet.md` 최신화(다음 task 명확히)
2. `skill-outputs/` 빈 TBD 제거 또는 의도적 보류 표시
3. DECISIONS·ASSUMPTIONS·OPEN_QUESTIONS 정리
4. `docs/PROJECT_TREE.md`·README 필요 시 추가
5. 알려진 제한·면책·연락처

## Output

`docs/agentic/skill-outputs/documentation-handoff.md` + 링크 목록
