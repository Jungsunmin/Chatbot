# Skill 공통 계약 (에이전트용)

## 시작 전

- [`docs/agentic/context_packet.md`](../../../docs/agentic/context_packet.md) 읽기
- 이전 단계 Skill 산출물이 `docs/agentic/skill_outputs.md`에 있는지 확인
- 확정되지 않은 내용은 **추측하지 말고** `ASSUMPTIONS.md` / `OPEN_QUESTIONS.md`에 기록

## 완료 후

1. `docs/agentic/skill_outputs.md` — 이 Skill 섹션 갱신 (TBD 제거)
2. `docs/agentic/context_packet.md` — `Current Next Task` 등 반영
3. 확정·가정·질문 — 각각 `DECISIONS` / `ASSUMPTIONS` / `OPEN_QUESTIONS`
4. `.cursor/hooks/refresh-workspace-tree.sh` 가 트리 스냅샷 갱신 (자동)

## 승인

서비스 목표·MVP·보안·배포 책임은 **사용자 승인** 후 다음 Skill로 진행 (`AGENTS.md`).
